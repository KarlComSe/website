JavaScript och MySQL
====================

Presentationen handlar om hur man kan koppla ett programmeringsspråk likt JavaScript till en databas som MySQL (eller MariaDB). Det visas kodexempel som utgår får Node.js och ett libb som sköter kopplingen och MySQL/MariaDB.

Denna föreläsning är bra att se på som en del i förberedelserna inför att jobba med kmom02.

I kursmomentet återkommer en övning där man gör detta praktiskt.

Videon är 16 minuter lång.
